# Stitching-Experiment

The files consist of data collection files:
- The solenoids.py file runs on a raspberry pi to control the solenoid circuit for the robot.
- The arduino code is for the data collection from a wire draw potentiometer and IMU.
- The serial program is a winform used to collect the serial data.

And also processing files:
- The Edit Serial Data python file, simplifies the serial output for processing.
- The cpp program performs the images stitcing.
